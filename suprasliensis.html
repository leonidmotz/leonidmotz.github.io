Welcome to this corpus viewer for Codex Suprasliensis, a by-product of the data analysis work needed for my MA thesis project on “Studies in Old Bulgarian Nominal Morphology: Stem Class Analogy in Codex Suprasliensis” (Supervisors: Prof. Roman N. Krivko, Dr. Ivan P. Petrov) at the University of Vienna’s Institute of Slavonic Studies.

The corpus uses the annotated XML from the TOROT/PROIEL project to which I added etymological stem class and declension type information.

You can navigate through the text using ◂ Prev and Next ▸, by scrolling continuously, or by choosing a folio in the drop-down menu. For the Foliation Guide, click the ⓘ button next to the folio selector. Clicking on a folio mark opens the corresponding folio in Birnbaum’s online edition.

The top header contains: a ☰ Lines button toggling between continuous and line view; a 🍐 toggle for TOROT/XML character variants; a абв toggle for Old Cyrillic view; and a font size drop-down menu.

The collapsible left-hand sidebar gives you refined search options, a dictionary, a stem class analogy meter, and a text part navigation menu. Double-clicking on a noun anywhere in the corpus opens a detailed morphology panel.

Please be aware that this is work in progress!
